# Internet Speed Test

A simple **iOS SwiftUI app** to measure your internet download speed.

## Features
- Downloads a test file to measure download speed (in MB/s)
- Records the date and measured speed in local history
- Displays current connection status (Wi-Fi or Cellular)
- Easy-to-read interface with results and progress indicator

## Usage
1. Open the app on your iPhone.
2. Tap "Start Test" to measure download speed.
3. View the results and check the history of previous tests.

## Notes
- The app stores results locally on the device.
- Test file is downloaded from the internet each time you run the test.
